# Smart Traffic Management System Using AI

## Description
The Smart Traffic Management System using AI is designed to optimize traffic flow by analyzing
real-time and historical traffic data. The system helps reduce congestion and improve road safety
by making intelligent traffic control decisions.

## Technologies Used
- Python
- Machine Learning
- AI Algorithms
- OpenCV 
- Flask 

## Features
- Traffic congestion detection
- Intelligent signal timing
- Vehicle density analysis
- Real-time traffic monitoring

## How to Run the Project
1. Install Python on your system
2. Clone or download this repository
3. Install required libraries:
   pip install -r requirements.txt
4. Run the main file:
   python app.py
